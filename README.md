# 🛒 Loja Virtual

Uma aplicação web simples de **loja virtual** desenvolvida em **HTML, CSS e JavaScript**, com suporte a **cadastro, login de usuários, modo claro/escuro e carrinho de compras**.  
O projeto foi criado para fins de aprendizado e demonstração de conceitos de front-end e autenticação básica.

---

## 🚀 Funcionalidades
- Cadastro e login de usuários (armazenados em `localStorage` para DEMO).
- Alternância entre **modo claro e escuro**.
- Layout **responsivo** para desktop, tablet e celular.
- Exibição de produtos em **cards** com imagem, descrição, preço e botão de compra.
- **Carrinho de compras** simples: adicionar, remover e visualizar itens.
- Estrutura inicial para futura integração com backend e banco de dados.
